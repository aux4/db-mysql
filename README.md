# aux4-db-mysql

MySQL database tools using mysql2 client for the aux4 ecosystem.

This package provides a MySQL database driver that follows the aux4 db driver specification outlined in [db/package/README.md](../db/package/README.md).

## Installation

```bash
npm install aux4-db-mysql
```

## Development

```bash
# Install dependencies
npm install

# Build the executable
npm run build

# Run in development mode
npm run dev
```
